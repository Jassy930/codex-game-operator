20M 后远航调度会让当前航段指定主动指令，并形成“目标 -> 协同/绕行 -> 回目标”的 3 步短循环；完成闭环时会触发远航闭环、按当前能量航段剩余量注入远航突破，并根据协同或绕行分支进入整备。目标后续走上一轮分支会触发“路线稳航 +4%”；选择与上一轮不同的分支会触发“分支改道 +6%”，随后回到目标完成 3/3 时还会触发“轮替闭环 +9%”；当前航段奖励类型还会决定“航段契合 +5%”：点击/过载航段契合协同，自动/总产能航段契合绕行；如果本轮按当前航段契合分支回到目标完成 3/3，还会触发“契合闭环 +7%”。主操作区会用分支选择条并排显示协同补给和绕行投送两条路线，并额外显示路线收益对照、路线目标、路线下一步、当前推荐分支、分支轮替、路线履历、路线预案、路线步骤和闭环复盘，把本步合计收益、回目标闭环收益、本轮操作意图、下一步按钮顺序、路线状态、资源取向、推荐原因、后续协同/绕行回航、路线稳航、航段契合、可改道奖励、轮替闭环预告、契合闭环预告、已记录路线、本轮路线、“目标 -> 分支 -> 回目标”的执行预案、当前第几步、刚完成路线和下一步整备合并到调度层；目标后的契合分支按钮会直接显示“推荐协同”或“推荐绕行”，并额外显示“路线 1/3 目标”“路线 2/3 推荐协同/绕行”“路线 3/3 回航”或整备/回航标记，执行后行动反馈会确认“路线执行”和“路线反馈”，说明刚完成的路线结果和下一步。星图当前航段卡片和远航路径轨会把第二步标成具体指令名，例如“协同 谐振脉冲 / 绕行 巡航回收”，鼓励在补当前资源、投送累计航段、后续回航结果、推荐原因、路线稳航、当前航段契合、契合闭环、收益对照、路线目标、路线下一步、路线预案、路线步骤、闭环复盘、按钮标记和路线反馈之间交替规划。

筛选摘要默认只显示短标签，例如“航段 1/57 点亮星图”；完整章节位置、后续和终点航段保留在悬停标题、可访问标签和展开详情中。

筛选航段抽屉里的按钮带有固定视觉标识，用小图形和颜色区分全部、当前、章节、累计/升级、奖励方向和完成状态入口。

章节视觉导航按钮带有微图景，用轨道、星门/信标和章节信号区分首段星图、专精校准、深空基建和远航长尾。
