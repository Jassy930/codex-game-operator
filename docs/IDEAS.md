# Ideas

1. 快速第一次升级
2. 下一目标提示
3. 过载点击
4. 离线能量：已完成离线收益结算与回收提示，后续可扩展离线奖励选择。
5. 升级可读性：已完成购买进度、剩余能量和可购买状态提示。
6. 目标可读性：已完成当前目标进度、剩余量和完成状态提示。
7. 过载可读性：已完成连击过载进度、剩余点击数和触发状态提示。
8. 目标行动提示：已完成购买型目标的关键升级能量缺口和可购买状态提示。
9. 操作反馈可见性：已完成点击收益、过载收益和升级购买结果提示。
10. 目标完成反馈：已完成目标切换时的完成提示和下一目标提示。
11. 目标升级推荐：已完成当前目标对应升级卡片的高亮和“目标推荐”标记。
