# Product

《星核工坊》是浏览器端数值增长放置类游戏。目标是让玩家 30 秒内获得第一次正反馈。

当前版本：

- 点击星核获得能量。
- 点火按钮会显示即时点击反馈：鼠标或触屏按下时先根据当前落点进入按住即时反冲，并让星核本体、蓄能外弧、中心标签和落点热区进入按住反馈；普通点击触发按钮脉冲、按压反冲、外层轮廓命中亮闪、中心标签命中亮脉冲、粒子层、蓄能轨命中闪烁、点击落点闪光、落点涟漪、落点火花束和本次 `+X` 收益浮层，收益浮层会从实际点击位置附近冒出且键盘触发回退到按钮中心；连击接近过载时显示蓄能前兆，过载触发时显示更强冲击波、按压反冲、过载外层轮廓亮闪、过载中心标签闪爆、过载蓄能轨余辉、过载落点闪光、过载涟漪、过载火花束和过载收益浮层。
- 点火按钮的星核 SVG 会显示随连击进度逐步显现的蓄能裂纹，过载前一击脉动，过载命中时爆发；该效果只从现有连击进度派生，不改变存档或收益。
- 点火按钮外层会显示随连击进度增强的蓄能外弧，连续点火时外弧逐步变亮、放大并加快旋转，过载前一击和过载命中时切换更强外弧状态；该效果只从现有连击进度派生，不改变存档或收益。
- 点火按钮提供可关闭的本地音效反馈：默认开启，普通点火播放轻双音，过载点火播放更厚的双音；音效偏好独立保存在浏览器本地，不改变游戏进度存档。
- 点火按钮提供可关闭的移动端触感反馈：默认开启，普通点火触发短震动，过载点火触发更强节奏；触感偏好独立保存在浏览器本地，不改变游戏进度存档。
- 点火按钮主体显示环形蓄能轨，随当前 8 格连击进度逐步点亮，并在普通点击命中时短促闪烁、过载命中时播放更强余辉，过载前兆和过载命中时高亮。
- 点火按钮主体显示阶段光环，随当前 8 格连击进度逐步升温，3/8 后进入预热、6/8 后进入升温、过载前一击脉动，过载命中时播放爆发态。
- 点火按钮内显示过载倒计时徽标，直接展示距离过载还剩几次，并在过载前一击和过载命中时高亮/爆发。
- 点火按钮内显示 8 格连击轨，标出当前连击、下一次点火和第 8 格过载点；连击轨会随当前连击进度显示连续填充，每次点火后本次命中的连击点会短促跳闪，第 8 格过载命中会使用更强跳闪，并在过载前一击和过载命中时加强填充色；按钮下方显示下一击奖励预告，临近过载时直接提示下一击会触发过载奖励，普通命中和过载命中后该预告会跟随按钮命中状态短促跳闪；下方连击读数会在普通命中时短促跳闪，过载命中时连击读数和过载状态读数会播放更强跳闪。
- 主操作区包含项目内 SVG 工坊插画，展示星核、聚能透镜、自动采集臂、稳定器和谐振环，减少点火主循环的纯文字与按钮占比。
- 连续点击会推进过载进度，第八次连击触发额外能量奖励。
- 星核谐振器可提升过载奖励，顶部产能区会显示当前过载奖励。
- 点击、过载和购买升级后，主操作区显示本次行动结果。
- 完成目标时，主操作区显示已完成目标和下一目标。
- 购买升级提升点击产能、自动产能和总产能倍率；升级卡片显示固定尺寸图标、成本、购买进度、剩余能量、可购买状态和目标推荐标记。
- 当前目标提示玩家追逐下一次正反馈，并显示进度、剩余量、关键升级能量缺口和完成状态。
- 100K 以后提供星图计划项目层，星图总览会显示已完成项目数、下一段航段序号、下一段目标、奖励、目标分轨、阶段导航、章节目标、章节奖励、奖励进度、奖励目标、里程碑、航线焦点、航线构成、奖励分布、已生效星图加成、行动建议和最多 3 个未完成项目的短摘要预告；完整三段名称与奖励预告保留在悬停标题和可访问标签中；星图视觉航线会把 57 段压缩成已完成、当前和待推进节点，并随当前筛选高亮命中航段；目标分轨会同时标出下一个累计航段和下一个升级航段，阶段导航会把 57 段星图拆成首段星图、专精校准、深空基建、远航长尾四个阶段，并显示当前阶段、章节内位置、全局航段号、本阶段剩余航段数和下一阶段名称，章节目标会按四个章节列出各自下一段未完成目标或已完成状态，章节奖励会按章节汇总总产能、点击、自动和过载奖励方向，奖励进度会按总产能、点击、自动和过载显示已生效奖励数/总奖励数，奖励目标会按总产能、点击、自动和过载列出下一条未完成奖励航段、奖励内容和当前进度，里程碑会显示当前章节终点和终局航点的航段序号、奖励与进度，航线焦点会按当前航线策略显示对应专精航段、奖励和进度，航线构成会显示累计/升级航段数量和四类奖励分布，行动建议会把当前航段转成可购买升级、能量缺口、继续点火/放置和预计时间；项目会显示航段序号、章节内位置、累计/升级航段标签、奖励类型标签、状态标签、进度、剩余量和奖励，列表默认进入本章筛选，标记第一个未完成项目为当前航段，当前航段保持完整详情展开，非当前航段默认折叠说明、进度、奖励和进度条，并支持按全部、当前、本章、累计、升级、总产能、点击、自动、过载、未完成和已完成筛选；筛选按钮会对内容视图显示已完成/总数，对当前、未完成和已完成视图显示直接数量；筛选区默认显示单行短摘要，包含当前筛选名、完成数、下一条全局航段和剩余航段数，完整筛选说明保留在悬停标题、可访问标签和展开详情中；完整详情默认折叠并保留当前视图的航段数、完成数、完成率、剩余航段数、状态构成、奖励构成、推进构成、章节构成、已领取奖励、待领取奖励、下一条可推进航段、后续最多两条未完成航段和筛选终点，其中下一条、后续和终点会同时显示全局航段号与章节内位置；下一个未完成项目会进入当前目标，主目标条会显示航段序号和项目奖励，完成项目时会提示已获得奖励，并自动提升有效产能或过载奖励。
- 星图项目卡片标题前显示两个固定尺寸图标，分别标识累计/升级推进方式和总产能/点击/自动/过载奖励方向，保留原有文字标签用于解释；卡片主体会显示 `project-card-scene` 可视化进度缩略图，用路线标记、进度填充、当前节点和奖励方向图形辅助扫视。
- 星图总览默认可见区域包含“当前航段视觉卡”，以固定视觉槽显示当前航段号、章节位置、累计/升级推进类型、奖励方向和进度条，减少从多行文本组合当前目标状态的成本。
- 星图总览默认可见区域包含“星图奖励罗盘”，以总产能、点击、自动和过载四条固定视觉槽显示对应奖励完成数、进度条、当前/待推进状态和下一条目标的可访问说明，减少读取奖励进度长文本的成本。
- 星图总览默认可见区域包含“航线预告视觉带”，以最多三条固定视觉槽显示后续未完成航段、推进类型、奖励方向和进度条；旁边的航线预告文本默认压缩为短摘要，完整三段名称与奖励说明保留在 `title` / `aria-label`。
- 星图筛选区提供首段星图、专精校准、深空基建和远航长尾四个章节入口；筛选按钮显示章节完成数，筛选摘要和星图视觉高亮会随章节切换同步更新。
- 星图视觉航线下方提供章节视觉导航，四个章节以可点击视觉按钮展示完成进度、下一条目标和小型进度轨，点击后直接切换到对应章节筛选。
- 章节视觉导航为四个章节提供不同徽记和焦点标签：星核点亮、策略校准、基建扩展、远航航线，降低四个章节入口只靠文字区分的压力。
- 章节视觉导航按钮内新增章节微图景，用固定轨道、星门/信标和章节信号图形区分首段星图、专精校准、深空基建和远航长尾，让章节入口比单纯文字按钮更容易扫视。
- 星图视觉航线标题下显示当前章节大图景，用更大的轨道、星门/信标、章节信号和进度条展示当前筛选章节或当前航段所属章节，让玩家切换章节时先看到画面锚点。
- 当前章节大图景内显示章节节点带，用已完成、当前、部分完成和待推进节点展示章节内推进节奏；远航长尾会压缩为最多 8 个节点，避免 44 段长尾再次形成密集节点墙。
- 星图筛选按钮默认收进“筛选航段”抽屉，首屏保留章节视觉导航和当前筛选摘要，展开后仍可访问全部筛选入口。
- 星图筛选按钮显示固定视觉标识，分别区分全部、当前、章节、累计/升级、奖励方向和完成状态入口，降低筛选抽屉展开后只靠按钮文字扫视的压力。
- 星图项目列表在当前筛选结果超过 8 段时，默认只显示包含当前航段的 8 段窗口，其余航段收进“已收起 X 段”的可展开区域，减少全部、未完成和远航长尾视图的一次性卡片数量。
- 星图总览默认只展开星图进度、当前进度、行动建议和航线预告短摘要；目标分轨、阶段导航、章节目标、章节奖励、奖励进度、奖励目标、里程碑、航线焦点、航线构成和生效加成放入可展开的“星图明细”，降低默认文字密度。
- 星图计划区域现在包含项目内 SVG 星域插画，展示星核、航线节点、远航星门和深空航迹，减少星图区纯文字占比。
- 100K 以后解锁星图航线策略，玩家可以在均衡航线、点火优先和巡航优先之间切换，分别保持默认节奏、强化点击/过载或强化自动产能；每个策略绑定一个契合指令，匹配当前策略的航线指令会获得“策略契合 +10%”。
- 航线策略按钮显示固定尺寸视觉徽记：均衡、点火、巡航三类策略分别使用不同图形和颜色，降低策略区只靠文字区分的压力。
- 100K 以后解锁航线指令，玩家可以主动执行点火齐射、巡航回收和谐振脉冲；指令带独立冷却，分别按当前点击、自动和过载收益立即结算能量。
- 航线指令按钮显示固定尺寸视觉徽记：点火齐射、巡航回收、谐振脉冲三类主动指令分别使用不同图形和颜色，降低指令区只靠文字标题和收益徽标区分的压力。
- 航线指令区包含项目内 SVG 插画，展示点火齐射、巡航回收、谐振脉冲和三步轮换轨，让 100K 后主动玩法入口先出现可扫视画面。
- 航线指令徽记右下角显示状态灯：可执行、冷却中和推荐态会在图标内直接变化，配合冷却进度轨降低按钮区只靠文字状态判断的压力。
- 航线指令按钮会压缩文字徽标和收益明细：运行期每个按钮最多直接显示 3 个文字徽标，其余折成 `+N`，预计收益行显示“预计 +X 能量 · N 项明细”，完整明细仍保留在悬停标题和可访问标签中。
- 航线指令会显示策略契合徽标：均衡航线契合谐振脉冲，点火优先契合点火齐射，巡航优先契合巡航回收；匹配时预计收益、执行反馈和本地事件都会记录这 10% 加成。
- 航线指令支持“策略终结”：完成 3/3 轮换时，如果第三步收束到当前航线策略契合指令，会额外获得基础指令收益 12% 的奖励，并在预计收益、执行反馈和本地事件中记录。
- 航线指令支持“航线连携”：90 秒内轮换不同指令会让下一次指令获得 +12% 或 +24% 连携加成，完成 3/3 轮换时还会获得基础指令收益 18% 的轮换目标奖励；重复同类或超时会重置。
- 航线指令支持“预案执行”：执行当前指令轮换预案推荐的指令时，额外获得有效基础指令收益 6%，预计收益、执行反馈和本地 `directive` 事件都会记录 `planReward` / `planBonusRate`，让按钮推荐从提示变成有即时收益的主动选择。
- 航线指令按钮会把“预案执行 +X”直接显示为奖励徽标，让推荐指令的额外收益不只藏在预计收益行和执行反馈里。
- 航线指令下方显示“航线委托”短期任务：100K 后按推荐预案推进 0/3 到 3/3，并用固定进度条显示当前委托进度；完成 3/3 推荐轮换时获得有效基础指令收益 8% 的“委托完成”奖励；按钮徽标、预计收益、执行反馈和本地 `directive` 事件会显示/记录 `taskReward` / `taskRewardRate`。
