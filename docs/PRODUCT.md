# Product

《星核工坊》是浏览器端数值增长放置类游戏。目标是让玩家 30 秒内获得第一次正反馈。

当前版本：

- 点击星核获得能量。
- 连续点击会推进过载进度，第八次连击触发额外能量奖励。
- 星核谐振器可提升过载奖励，顶部产能区会显示当前过载奖励。
- 点击、过载和购买升级后，主操作区显示本次行动结果。
- 完成目标时，主操作区显示已完成目标和下一目标。
- 购买升级提升点击产能、自动产能和总产能倍率；升级卡片显示成本、购买进度、剩余能量、可购买状态和目标推荐标记。
- 当前目标提示玩家追逐下一次正反馈，并显示进度、剩余量、关键升级能量缺口和完成状态。
- 100K 以后提供星图计划项目层，星图总览会显示已完成项目数、下一段航段序号、下一段目标、奖励、已生效星图加成和最多 3 个未完成项目的名称与奖励预告；项目会显示航段序号、进度、剩余量和奖励，列表会标记第一个未完成项目为当前航段；下一个未完成项目会进入当前目标，主目标条会显示航段序号和项目奖励，完成项目时会提示已获得奖励，并自动提升有效产能或过载奖励。
- 100K 以后解锁星图航线策略，玩家可以在均衡航线、点火优先和巡航优先之间切换，分别保持默认节奏、强化点击/过载或强化自动产能。
- 星图计划已扩展到 22 段，包含点火航校、巡航航校和均衡校准三段策略专精目标，把航线策略对应到可追进度和长期奖励。
- 航线策略按钮会显示对应策略专精航段的当前进度或完成状态，让策略选择和后续追逐目标保持同屏关联。
- 点亮星图后新增谐振校准目标，要求星核谐振器达到 6 级并提升过载奖励，给中后段玩家一条不同于常规产能倍率的成长线。
- 250K 后继续提供稳定矩阵和远星中继长尾项目；500K 后继续提供深空矿带、星环工厂和恒星锚点，1.5M 后提供星门远征，2M 后提供外环信标，2.75M 后提供彗尾船坞，3.5M 后提供引力航标，4.5M 后提供暗流观测站，5.75M 后提供虚空罗盘，7.25M 后提供星云档案库，9M 后提供深穹测距阵，11M 后提供寂光中继环，延长星图计划的追逐目标。
- 玩家回到页面时自动结算离线收益，并在主操作区显示回收提示。
- 侧栏反馈表单可生成预填 GitHub Issue，并附带包含过载奖励和航线策略的游戏快照。
