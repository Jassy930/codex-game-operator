## 2026-04-30 航线指令轮换目标

本轮未新增外部网页调研；依据来自真实 GitHub 反馈 #3、上一轮“航线连携”的效果边界和当前主操作区复盘。三个指令已有收益、冷却和连携加成，但玩家还需要自己判断连携是否仍在窗口内、当前是第几步、下一步应该切到哪一个不同指令。

产品结论：在现有指令下方补一个“指令轮换”目标提示，比新增第四个指令、改收益曲线或增加新资源更小、更可验证。它把 90 秒窗口、0/3 到 3/3 进度和下一步建议直接展示出来，只从现有冷却与连携字段推导，不新增存档字段或改变数值。

## 2026-04-29 星图视觉航线

本轮未新增外部网页调研；依据来自真实 GitHub 反馈 #4 和当前界面复盘。#4 明确指出界面“全是密密麻麻的文字”，并希望生成图片。当前星图区已经堆叠大量总览、章节、奖励、筛选和项目文本，继续增加说明会加重反馈中的密度问题。
