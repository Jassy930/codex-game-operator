# Research

Agent 可以主动上网调研 idle/incremental/clicker game 设计和留存问题。

## 2026-05-01 点火落点涟漪

本轮未新增外部网页调研；依据来自真实 GitHub 反馈 #5 和当前点火按钮实现复盘。点火按钮已经具备脉冲、粒子、收益浮层、蓄能轨、阶段光环、倒计时徽标、落点闪光、星核裂纹和蓄能外弧，但点击落点反馈仍主要是一枚亮点。连续点击时，玩家可以获得更明确的“按下处扩散出能量环”的局部反馈。

产品结论：复用现有点击坐标 `--core-impact-x` / `--core-impact-y`，新增 `core-impact-ripple` 并由 `animateCore` 同步触发，比继续提高收益、增加说明文字或新增存档字段更小、更可验证。该方案只改变展示层；完整连击与过载语义仍保留在连击轨、过载倒计时徽标、蓄能裂纹、蓄能外弧、下一击预告、按钮状态和行动反馈中。

## 2026-05-01 点火蓄能外弧

本轮未新增外部网页调研；依据来自真实 GitHub 反馈 #5 和当前点火按钮实现复盘。点火按钮已经具备多层点击反馈和星核内部裂纹，但连续点击时按钮外层轮廓仍主要依赖环形蓄能轨与阶段光环，缺少一个能随连击进度持续加速的外层运动信号。
